# Third-party notices

Dungeon Defense 0.1.0 does not intentionally ship third-party artwork, audio,
or runtime libraries as game assets. Its source toolchain dependencies are
recorded in `package.json` and `pnpm-lock.yaml`.

Before any public release, add the required licence and attribution notices for
each bundled dependency or third-party asset. Keep this file with distributed
source and release archives when notices become applicable.
