# Licence selection required

No project licence has been selected. This repository is therefore not ready
for public distribution or reuse outside the permissions granted directly by
the project owner.

Before publishing a release, the owner must:

1. Choose the intended permission and contribution model.
2. Add the selected licence text in a root `LICENSE` file.
3. Confirm that contributor, asset, and dependency obligations are compatible
   with that licence.
4. Update `NOTICE.md` with any required third-party attributions.
5. Change the release checklist and README to identify the selected licence.

This document deliberately does not recommend a licence: the appropriate
choice depends on ownership, intended distribution, and legal requirements.
